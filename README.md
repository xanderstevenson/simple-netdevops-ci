# simple-netdevops-ci

## This code is meant to accompany the DevOps Shop video titled 'Simple NetDevOps CI with GitHub, Jenkins, pytest, and Webex'.

The code works with:

Jenkins 2.401.3
pytest 7.4.0
Python 3.9.5
Webex 43.7.0.26612
